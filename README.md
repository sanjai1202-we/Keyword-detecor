# SDN AR Scanner

Scan the SDN Controllers textbook page → video plays automatically.

## Files
- `index.html` — main AR scanner page
- `video.mp4` — SDN Controller video (compressed for mobile)

## How it works
Camera reads text on the page using OCR. When it detects keywords like
"SDN Controller", "OpenFlow", "Flow Management" etc — video plays automatically.

## Works for everyone
No image tracking file needed. Works at any angle, any lighting, any phone.
